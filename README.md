## 🛠️ 功能详解

### 🔐 Windsurf 机器码重置
**解决设备限制问题，重置 Windsurf 机器标识符。**

- **🔄 重置机器码**：生成新的设备标识符。
- **📋 查看详细信息**：查看当前机器码。
- **⚠️ 注意**：重置后需重启 Windsurf 生效。

### 🚀 Windsurf 无感换号 - 号池定制版
**专为 Windsurf 设计的账号管理与无感切换工具。**

- **⚡ 无感切换**：一键切换账号，无需任何手动登出/登录操作。
- **📊 号池集成**：支持通过激活码/后端 URL 自动获取可用账号。
- **⚖️ 智能路由**：自动检测账号用量，点数耗尽智能提示。
- **⌨️ 效率操作**：支持全局快捷键（`Ctrl+Alt+S`）进行账号轮转。
- **🛡️ 隐私保护**：全过程本地化处理敏感信息。

## 📦 安装方法

1. 下载对应的 `.vsix` 文件。
2. 在 Windsurf/VS Code 中打开 **Extensions (扩展)** 视图。
3. 点击右上角 `...` -> **Install from VSIX...**。
4. 选择文件并安装，完成后**重启编辑器**。
