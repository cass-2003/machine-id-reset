# Windsurf 机器码重置工具

**一键重置 Windsurf 机器标识符**

---

## 📥 下载安装

### 直接下载 VSIX

👉 **[点击下载最新版 machine-id-reset-1.0.1.vsix](https://raw.githubusercontent.com/cass-2003/machine-id-reset/main/machine-id-reset-1.0.1.vsix)**

### 安装方法

在 Windsurf/VS Code 中：
1. 扩展 → 从 VSIX 安装
2. 选择下载的 `.vsix` 文件
3. 重启编辑器

---

## 功能

安装后左侧边栏会出现 🔐 图标，点击即可：

- **🔄 重置机器码** - 生成新的设备标识符
- **📋 查看详细信息** - 查看当前机器码
- **↻ 刷新** - 刷新当前信息

也可通过命令面板（`Ctrl+Shift+P`）执行：
- `Windsurf: 重置机器码`
- `Windsurf: 显示当前机器码`

---

## 注意事项

⚠️ **运行前请关闭 Windsurf**

重置后需要重启 Windsurf 才能生效。
